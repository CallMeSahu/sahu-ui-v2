Image

Responsive Image Components are used to display Images on websites, which are screen responsive. They expand upto the width of the container they are in.

Different Image Types

Images can be used by including the keyword in the class followed by its varient name. There are 2 varients of image available which are responsive-image & rounded-image. Changing the varient name affects the image shape.
