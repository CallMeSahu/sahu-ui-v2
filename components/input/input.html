Input

Inputs are necessary for user interaction with the website. Inputs are used to take information from the user, and then save the data to the server, accordingly.

Labelled Inputs

Labbeled Inputs can be used by including the input keyword in the class followed by its varient name. There are 3 varients of labelled inputs available which are TEXT, NUMBER & PASSWORD. Changing the varient name affects the ability to take a particular input type.

Radio and Checkbox Inputs

Radio and Checkbox type inputs can be used by including the radio or checkbox keyword in the class. If you want your input to be disabled don't forget to add disabled keyword to your input tag.
