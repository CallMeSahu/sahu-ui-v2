Avatar

Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects projects, spaces, repositories to give them a recognisable visual identity.

Different Size Types

Avatar can be used by including the avatar keyword in the class followed by its varient name. There are 3 varients available which is SMALL, MEDIUM & LARGE. Changing the varient name affects the avatar size.
