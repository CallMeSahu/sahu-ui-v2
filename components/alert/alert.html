Alert

An alert displays a short, important message that attracts the user's attention without interrupting the user's task.

Different Alert Varients

Alerts can be used by including the alert keyword in the class followed by its varient name. There are 4 varients available which is SUCCESS, WARNING, INFO & DANGER. Changing the varient name affects the colour scheme.
