Button

Buttons are the soul of an interactive Web Application. Buttons allow users to take actions, and make choices, with a single tap.

Standard Buttons

Standard Buttons can be used by including the btn keyword in the class followed by its varient name. There are 7 varients of standard button available which are PRIMARY, SECONDARY, SUCCESS, DANGER, WARNING, DEFAULT & LINK. Changing the varient name affects the button colour scheme.

Link Button

Outlined Buttons

Outlined Buttons can be used by including the btn keyword in the class followed by outlined & its varient name. There are 7 varients of standard button available which is PRIMARY, MSECONDARY, SUCCESS, DANGER, WARNING, DEFAULT & LINK. Changing the varient name affects the button colour scheme.

Link Button

Iconed Buttons

Iconed Buttons can be used by including the btn-icon keyword in the class followed by its varient name. There are 7 varients of standard button available which are SHOPPING-CART, TRASH, COMMENT, SHARE, SHARE-ALT, HOME & MENU. Changing the varient name affects THE icon scheme.

Floating Buttons

Floating Buttons can be used by including the btn-float keyword in the class followed by its varient name. There are 2 varients of floating button available which are ADD & SCROLL-UP. Changing the varient name affects the icon scheme of the button.
